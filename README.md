# What's This?

**WARNING: this code is not yet functional!**

A SAFE Network web API with application, file system and RESTful services for NodeJS applications.

For web/browser applications, see [safenetwork-web](https://github.com/theWebalyst/safenetwork-web) which is a version of this library built for use in the browser with additional web specific features (including RESTful services for SAFE Network, implemented in the client).
